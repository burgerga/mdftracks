mdftracks
=========

Reads the '.mdf' files generated by MTrackJ and convert it to a data.frame.

Example usage:

``` r
library(mdftracks)
data <- read.mdf('~/mdftracks.mdf', drop.Z = T)
head(data, 10)
```
